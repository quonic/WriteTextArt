## Installation
WriteTextArt is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/WriteTextArt/).

To Inspect:
```powershell
Save-Module -Name WriteTextArt -Path <path>
```
To install:
```powershell
Install-Module -Name WriteTextArt -Scope CurrentUser
```
